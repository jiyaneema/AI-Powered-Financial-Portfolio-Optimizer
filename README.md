This project is a full-stack web application that helps users optimize their financial portfolios using AI and machine learning. It provides:

- **Frontend:** React (with Tailwind CSS) for a modern, responsive UI.
- **Backend:** FastAPI (Python) for API endpoints and business logic.
- **Machine Learning:** scikit-learn for portfolio optimization algorithms.
- **External APIs:**
  - **Alpaca API** for real-time financial data.
  - **OpenAI GPT-4** for natural language explanations and insights.
- **Database:** In-memory or dummy data for initial development (can be replaced with a persistent database later).

## Structure

- `my-app/` — Frontend (React + Tailwind CSS)
- `backend/` — Backend (FastAPI + scikit-learn)

## Getting Started

### Frontend
```
cd my-app
npm install
npm run dev
```

### Backend
```
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
